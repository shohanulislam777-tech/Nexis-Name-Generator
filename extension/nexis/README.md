# Nexis AI Tool — Chrome Extension

A next-generation Chrome Extension for lovable.dev with enterprise licensing.

## Installation

1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer Mode** (top right toggle)
3. Click **Load unpacked**
4. Select the `extension/nexis` folder
5. The Nexis extension will appear in your toolbar

## Configuration

Before loading, update the `NEXIS_API_BASE` URL in these files:
- `license.js` — line 9
- `content.js` — line 5
- `sidepanel.js` — line 3

Set it to your deployed Nexis backend URL, e.g.:
```
const NEXIS_API_BASE = "https://your-app.replit.app/api";
```

## Features

- **License Gate** — First-launch activation screen with key entry
- **Hardware Fingerprinting** — Multi-factor device fingerprint (WebGL, Canvas, Audio, Fonts)
- **Device Binding** — Automatic device registration on activation
- **License Heartbeat** — Periodic revalidation every hour
- **Status Screens** — Expired / Suspended / Revoked states
- **Side Panel Mode** — Persistent sidebar on lovable.dev
- **Floating Mode** — Draggable overlay on lovable.dev
- **Notifications** — Pull from Nexis admin panel
- **Sound Effects** — Error audio feedback
- **Prompt Templates** — Quick AI prompt shortcuts

## License Key Format
